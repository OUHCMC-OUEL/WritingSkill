# WritingSkill
Một nhánh nhỏ của hệ thống, làm về rèn luyện kỹ năng viết

# Báo cáo 29/11/2025

# Cách chạy dự án
1. Mở git bash và clone dự án từ github:
   ```git clone https://github.com/OUHCMC-OUEL/WritingSkill.git```
2. Chạy file run.sh để setup dự án
   ```./run.sh```
3. Đường dẫn truy cập giao diện và backend api:
   - Frontend: https://localhost:5173
   - Backend: https://localhost:8000
    
# Công cụ sử dụng
| Phần       | Công cụ |
|:---------- | ----:|
| front-end      | reactjs |
| back-end        | django   |
| css        | taiwind + antdesign   |
| AI       | gemini 2.5 flash  |
| deploy       | Vite   |
